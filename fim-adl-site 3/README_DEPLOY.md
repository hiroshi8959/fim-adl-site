# 運動FIM ADL予後予測サイト 公開手順

このフォルダは静的サイトです。サーバー処理や外部APIを使わないため、公開後もトークン消費はありません。

## 公開に含めるファイル

- `index.html`
- `styles.css`
- `app.js`
- `_headers`

## 一番簡単な公開方法

Netlify Drop や Cloudflare Pages などの静的ホスティングに、この `fim-adl-site` フォルダをアップロードしてください。

公開後に発行されるURLを共有すれば、URLを知っている人がブラウザから利用できます。

## 検索避けについて

`index.html` と `_headers` に `noindex, nofollow, noarchive` を入れています。検索エンジンに載りにくくするための設定です。

注意: これはパスワード保護ではありません。URLを知っている人はアクセスできます。

## 入力データの扱い

計算はすべてブラウザ内で完結します。入力された日付やFIM点数は外部サーバーへ送信されません。
